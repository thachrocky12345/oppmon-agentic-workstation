# Skill: repo-orientation-and-reference-docs

Description: Use this when you need a fast map of the repository and trusted docs.

## Instructions
- Read the repository’s agent instructions file first (e.g., `AGENTS.md`).
- Scan top-level folders to identify primary apps/services and docs.
- Use `ContextFiles/Skills/repo-orientation-and-reference-docs/references/` indexes to jump to sources.
- Prefer linking to source files over duplicating content in new notes.
- When new docs are added, update the relevant index and the audit map.

Hints / next steps: Add missing docs to `ContextFiles/Library/` and link them from the indexes.
